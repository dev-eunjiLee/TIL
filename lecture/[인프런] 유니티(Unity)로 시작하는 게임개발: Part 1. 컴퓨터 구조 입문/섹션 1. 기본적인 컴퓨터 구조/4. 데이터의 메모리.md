### 메모리의 사용
- 예시: 영상 파일을 재생
  - 영상 재생 파일을 더블 클릭 > 영상을 재생하는 프로그램과 파일이 메모리에 올라감 > 메모리에서 사용 > 종료 후 메모리에서 사라짐
- 실행 시 프로그램과 데이터가 메모리에 올라갔다가 사라짐
  - 데이터는 원본 데이터가 사용되는 것이 아니라 복사된 데이터가 메모리에 올라가고 사용 후 사라진다


### 메모리의 복사
- 복사와 이동
  - 복사: 원 데이터는 그대로 둔 채 복사하여 메모리에 올려 사용
  - 이동: 원 데이터를 복사한 후 원본을 삭제하는 것이 이동의 원리 
- 복사의 원리는 데이터가 같아지는 것이며, 이동은 복사를 하고나서 원본을 지우는 것

### 데이터의 메모리
- 데이터를 복사해 메모리에 생기고, 메모리에서 사용한 후 메모리에서 사라진다.
- 이 때 데이터가 올라가는 위치를 "데이터의 메모리"라고 함

## 정리
- 데이터 메모리: "데이터"를 사용하기 위해 메모리에 지정해놓은 영역
  - 프로그램 실행시 데이터 메모리에 데이터가 생겨나고 사용되고 사라진다
